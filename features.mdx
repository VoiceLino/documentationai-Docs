## Capture and Logging

VoiceLine's Capture feature records sales calls and automatically logs key details like duration, participants, and outcomes directly to your CRM. You focus on selling while the AI handles data entry.

### Set Up Automatic Logging

<Steps>
  <Step title="Connect Your CRM" icon="link">
    In the VoiceLine dashboard, navigate to Integrations and select your CRM provider.

    ```bash
    # Example API setup for Salesforce
    curl -X POST https://api.example.com/integrations/salesforce \
      -H "Authorization: Bearer YOUR_TOKEN" \
      -d '{"client_id": "YOUR_CLIENT_ID"}'
    ```
  </Step>
  <Step title="Enable Call Recording" icon="mic">
    Toggle on automatic recording in Call Settings. Ensure compliance with local recording laws.
  </Step>
  <Step title="Test Logging" icon="play">
    Make a test call and verify the log appears in your CRM within seconds.
  </Step>
</Steps>

Analyze turns raw call data into actionable market insights. VoiceLine detects sentiment, objections, and trends across your team's interactions.

    ```javascript
    // Fetch analysis via API
    const insights = await fetch('https://api.example.com/v1/calls/{callId}/insights', {
      headers: { 'Authorization': 'Bearer YOUR_TOKEN' }
    }).then(res => res.json());

    console.log(insights.sentiment); // e.g., { positive: 0.85, negative: 0.15 }
    ```
