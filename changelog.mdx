## Recent Updates

Stay up to date with the latest releases. VoiceLine releases new versions regularly to enhance AI analysis, capture tools, coaching features, CRM integrations, and mobile performance.

<Update label="2024-10-15" description="v1.5.0" tags={["feature", "improvement"]}>
## New Features

- Added advanced AI sentiment analysis for sales calls, detecting customer objections in real-time.
- Introduced CRM integration with Salesforce, automatically syncing call notes and outcomes.

## Improvements

- Enhanced mobile app performance, reducing load times by 40% on iOS and Android devices.
- Improved coaching module with personalized rep feedback based on call patterns.

## Bug Fixes

- Fixed capture module crashes during long recordings.
- Resolved sync issues with HubSpot CRM.

</Update>

const webhook = {
  url: 'https://api.salesforce.com/webhook',
  event: 'call_ended',

fetch('https://api.voiceline.com/webhooks', {

````python
import requests

webhook = {
    'url': 'https://api.salesforce.com/webhook',
    'event': 'call_ended',
    'secret': 'YOUR_WEBHOOK_SECRET'
}

requests.post('https://api.voiceline.com/webhooks',
              json=webhook,
              headers={'Authorization': 'Bearer YOUR_API_KEY'})
````
