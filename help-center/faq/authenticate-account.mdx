## Overview

To access VoiceLine, you need to log in using your registered email.

## Steps

1. Go to `https://dashboard.voiceline.ai`
2. Click "Log In"
3. Enter your email and password
4. Check your inbox for a code if prompted
5. Enter the code to complete login

## Tip

If you forgot your password, use the "Forgot Password" link to reset it safely.
